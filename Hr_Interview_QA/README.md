# 👥 HR Interview Preparation

> Everything you need to confidently pass the **HR and behavioural rounds** of a DevOps interview — from self-introduction to salary negotiation.

---

## 📌 Purpose

The HR round is often the **first filter** in a DevOps hiring process. This folder prepares you for every type of non-technical question you are likely to face, organised into 9 progressive sections plus specialist guides.

---

## 📂 Files in This Folder

### 🗣️ Fresher HR Q&A — 9 Sections

| File | Topics Covered |
|---|---|
| `section1-basic-intro.md` | Self-introduction, background, why you chose DevOps |
| `section2-career-goals.md` | Short-term goals, 5-year plan, career aspirations |
| `section3-strengths-weaknesses.md` | SWOT-style answers, examples, recovery strategies |
| `section4-team-work-behavior.md` | Collaboration, conflict resolution, communication |
| `section5-salary-commitment.md` | Salary expectations, notice period, commitment |
| `section6-devops-specific-hr.md` | DevOps culture, Agile mindset, continuous learning |
| `section7-situational.md` | "Tell me about a time when…" scenario questions |
| `section8-personality-attitude.md` | Attitude, adaptability, learning style |
| `section9-closing.md` | Smart questions to ask the interviewer at round's end |

### 📋 Specialist Guides

| File | Purpose |
|---|---|
| `hr-interview-Q&A-salary-negotiation.md` | Quick salary negotiation Q&A |
| `hr-interview-Q&A-salary-negotiation-guide.md` | Step-by-step salary negotiation tactics with scripts |
| `hr-interview-how-to-handle-unknown-Q&A-readme.md` | Strategies when you don't know the answer |
| `hr-professional-resume-email-readme.md` | How to write professional follow-up and intro emails |

---

## 📖 Suggested Usage

1. **1–2 days before an interview** → Work through sections 1–4 out loud
2. **Night before** → Review sections 5–9 and the salary negotiation guide
3. **Morning of** → Re-read section 1 (self-intro) and section 9 (closing questions)
